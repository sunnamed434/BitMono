BitMono Combo
=============

BitMono alone isn't the whole story. Its packers are genuinely good at keeping dnSpy and other popular
tools from even opening your file :D but for serious protection you'll still want another layer on top of
what BitMono does today.

The trick is the order: run the other tools first, and **BitMono last**. So a typical combo looks like:

1. ConfuserEx
2. any other tools you have and want to use
3. BitMono (last)

That way BitMono's packers wrap up everything the earlier tools produced.
